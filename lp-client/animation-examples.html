文字・図形の出現アニメーション例

1. 基本的なスクロールアニメーション

要素が画面に入った時にフェードイン + スライドアップします。

スクロールして表示される要素
<div class="animate-on-scroll">
    スクロールして表示される要素
</div>

2. 方向別アニメーション

様々な方向から要素が出現します。

下から上へ (fade-in-up)
上から下へ (fade-in-down)
左から右へ (fade-in-left)
右から左へ (fade-in-right)
<div class="animate-fade-in-up">下から上へ</div>
<div class="animate-fade-in-down">上から下へ</div>
<div class="animate-fade-in-left">左から右へ</div>
<div class="animate-fade-in-right">右から左へ</div>

3. スケール・回転アニメーション

要素が拡大・回転しながら出現します。

スケールアニメーション
回転アニメーション
<div class="animate-scale-in">スケールアニメーション</div>
<div class="animate-rotate-in">回転アニメーション</div>

4. 遅延アニメーション（段階的出現）

要素が順番に出現します。

1番目 (0.1s遅延)
2番目 (0.2s遅延)
3番目 (0.3s遅延)
<div class="animate-fade-in-up animate-delay-100">1番目</div>
<div class="animate-fade-in-up animate-delay-200">2番目</div>
<div class="animate-fade-in-up animate-delay-300">3番目</div>

5. 文字アニメーション

文字が一文字ずつ出現します。

一文字ずつ出現
タイプライター効果
<div class="animate-letters">一文字ずつ出現</div>
<div class="typewriter">タイプライター効果</div>

6. 図形アニメーション

図形が様々な方法で出現します。

円形のスケールアニメーション
バウンスアニメーション
パルスアニメーション
<div class="demo-shape animate-scale-in"></div>
<div class="demo-shape animate-bounce"></div>
<div class="demo-shape animate-pulse"></div>

7. SVG描画アニメーション

SVGのパスが描画されるように出現します。

SVG描画アニメーション
<svg class="animate-draw" viewBox="0 0 200 200">
    <circle cx="100" cy="100" r="80" fill="none" stroke="#00AFCC" stroke-width="4"/>
    <path d="M50 100 L90 140 L150 60" fill="none" stroke="#d63384" stroke-width="4"/>
</svg>

使用方法

1. 基本的な使い方

HTML要素にクラスを追加するだけでアニメーションが適用されます。

2. スクロール連動アニメーション

animate-on-scroll クラスを使用すると、要素が画面に入った時に自動的にアニメーションが実行されます。

3. 即座に実行されるアニメーション

animate-fade-in-up などのクラスを使用すると、ページ読み込み時に即座にアニメーションが実行されます。

4. カスタマイズ

CSSファイルでアニメーションの速度、遅延、イージングを調整できます。

5. JavaScript API

JavaScriptから動的にアニメーションを制御することも可能です：

// 要素をアニメーション
animateElement(element, 'animate-fade-in-up', 600, 200);

// 複数要素を順番にアニメーション
animateElementsSequentially(elements, 'animate-fade-in-up', 200);
