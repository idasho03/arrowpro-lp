FEATURE

即戦力コンサルタント特化のマッチングプラットフォーム

ARROW proは、昨今市場が急速に拡大している「コンサルタント」という職種に特化した、フリーランスと企業のダイレクトマッチングプラットフォームです。
登録者は著名コンサルティング会社出身のフリーランス、法人代表を中心に構成されています。高い専門性とタフな経験に裏打ちされた即戦力人材が、貴社の経営課題を支援します。

FEATURE 01
ハイクラスコンサルタントのデータベースから直接検索
大手コンサル出身者を中心とした人材データベースから、スキルや経歴、評価などで自由に検索できます。
FEATURE 02
最短即日のスピーディーな参画
プロフェッショナルへのコンタクトから契約まで、第三者の介入なく直接対話できるため、面談や手続きも迅速に進められます。
FEATURE 03
仲介手数料なしで直接マッチングし放題
プロフェッショナルと直接契約できるため仲介手数料がかからず、コストを抑えられます。

FAQ

よくある質問

Q
プロフェッショナルとの契約体系は？
▼
A
企業様とプロフェッショナルの二者間で、本サービス外で業務委託契約を締結いただきます。
Q
プロフェッショナルと契約に入ってもらうことはできますか？
▼
A
基本的には二者間での契約をお願いしております。当社経由でのご契約を希望される場合はご相談ください。
Q
正社員として採用オファーすることはできますか？
▼
A
業務委託での稼働開始を前提とした機能を提供しており、弊社としてあっせん・仲介などは行っておりません。業務委託の延長として雇用契約に移行いただくことは双方合意のもとご自由に行っていただけます。
Q
企業名や案件の機密性は保たれますか？
▼
A
はい、ご安心ください。ご登録企業様の名称は匿名での公開を設定でき、案件に関する情報も任意の情報量で公開が可能です。
プロフェッショナルと直接メッセージのやり取りを行う際に、企業様の判断で情報を開示していただく仕組みです。
